# Manuscript Workspace

Primary home for paper writing.

## Layout

- `main/`: core manuscript source files
- `appendix/`: supplementary proofs/derivations/results
- `output/`: rendered artifacts (PDF/HTML/DOCX if tracked)

## Integration rules

- Analysis code lives in `scripts/` and `research/replications/`.
- Shared figures live in `assets/figures/`.
- Citations are centralized in `assets/bibliography/references.bib`.
- Decisions/specs/logs go in `quality_reports/`.
