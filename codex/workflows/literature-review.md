# Literature Review Workflow (Codex)

Research-grade literature review workflow with explicit evidence tracking.

## Steps

1. Create or use `explorations/lit_review_<topic>/`.
2. Define query and inclusion/exclusion criteria.
3. Collect sources from:
   - `master_supporting_docs/supporting_papers/`
   - bibliography files (`*.bib`)
   - web search (when available)
4. Build an evidence table with:
   - question, data, method, identifying assumptions, key estimate, limitation
5. Synthesize by themes:
   - theoretical contributions
   - empirical findings
   - methodological innovations
   - unresolved debates
6. Produce:
   - gaps/opportunities
   - concrete next steps
   - BibTeX entries with confidence flags if uncertain
7. Save report to `quality_reports/lit_review_<topic>.md`.

## Required Files

- `explorations/lit_review_<topic>/papers.md`
- `explorations/lit_review_<topic>/synthesis.md`
- `quality_reports/lit_review_<topic>.md`
